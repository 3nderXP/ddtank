# DDTank Paraguai

Projeto feito em HTML5 e JavaScript, totalmente inspirado no famoso DDTank, um jogo estilo Gunbound.

## Tecnologias, padrões/paradígmas utilizados no projeto:
- `HTML5`
- `JavaScript`
- `Phaser3 (Biblioteca JS)`
- `PHP 8.2^`
- `Orientação a objetos`
- `Padrão MVC`
- `Componentização`
- `Sistema de rotas`
- `API`
